# Video Indir

macOS için video/ses indirme + transkript aracı (Apple Silicon).

## Kurulum

1. [Releases](https://github.com/uveyscolak/Video-Indir-Releases/releases/latest) sayfasından son DMG'yi indirin.
2. DMG'yi açın, **Video Indir**'i **Applications** klasörüne sürükleyin.
3. İlk açılışta "geliştirici doğrulanamadı" uyarısı çıkarsa (bir kereye mahsus):
   - Uygulamaya **sağ tık → Aç → Aç**, veya
   - **Sistem Ayarları → Gizlilik ve Güvenlik → "Yine de Aç"**

## Özellikler

- YouTube, Instagram, Udemy ve yüzlerce siteden video/ses indirme (kalite + codec seçimi)
- Kuyruk sistemi: birden çok URL sırayla iner, tek tıkla iptal
- Transkript: dosyadan veya linkten, txt / srt formatında
- Otomatik güncelleme bildirimi

## Transkript için (isteğe bağlı)

Ayarlar sekmesine ücretsiz Gemini API anahtarınızı girin: https://aistudio.google.com/apikey
Video indirme için anahtar gerekmez.
