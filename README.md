# VidScribe

macOS için video/ses indirme + transkript aracı (Apple Silicon).

## Kurulum

1. [Releases](https://github.com/uveyscolak/VidScribe-Releases/releases/latest) sayfasından son DMG'yi indirin.
2. DMG'yi açın, **VidScribe**'ı **Applications** klasörüne sürükleyin.
3. İlk açılışta "geliştirici doğrulanamadı" uyarısı çıkarsa (bir kereye mahsus):
   - Uygulamaya **sağ tık → Aç → Aç**, veya
   - **Sistem Ayarları → Gizlilik ve Güvenlik → "Yine de Aç"**

## Özellikler

- YouTube, Instagram, Udemy ve yüzlerce siteden video/ses indirme (kalite + codec seçimi)
- Kuyruk sistemi: birden çok URL sırayla iner, tek tıkla iptal
- **Transkript — kurulumsuz:** gömülü Whisper ile offline çalışır, anahtar gerekmez (ilk kullanımda model bir kez indirilir)
- **Transkript — Gemini (isteğe bağlı):** ücretsiz API anahtarı girersen bulutta daha hızlı/kaliteli sonuç
- txt / srt formatı, linkten veya dosyadan
- Otomatik güncelleme bildirimi
